# Presentation Materials

This folder contains the prepared materials for the 5-7 minute Assignment 4 demonstration.

- `slides.md` - slide-by-slide structure with speaker notes.
- `demo-video-script.md` - recording script, demo order, and timing plan.

Recommended final export:

1. Convert `slides.md` into slides using PowerPoint, Google Slides, Keynote, or Marp.
2. Export the slide deck as `slides.pdf`.
3. Record the live walkthrough using `demo-video-script.md`.
4. Save the recording as `demo-video.mp4`.
